RAX - About Author Widget is useful for site owners to show thier own profile as well as show their site contributer. This will add a sidebar widget in your wordpress blog as per details provided from wordpress admin. You can choose options from wordpress widget admin panel to add photo, align photo, add website url, add follow twitter profile and many more.

== Description ==

RAX - About Author Widget is useful for site owners to show thier own profile as well as show their site contributer. This will add a sidebar widget in your wordpress blog as per details provided from wordpress admin. You can choose options from wordpress widget admin panel to add photo, align photo, add website url, add follow twitter profile and many more.

== Installation ==

1. Download the zip folder named rax-about-author.zip

2. Unzip the folder and put it in the plugins directory of your wordpress installation. (wp-content/plugins).

3. Activate the plugin through the plugin window in the admin panel.

4. Go to widget and select Rax-About Author widget.
